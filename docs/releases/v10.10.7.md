# GPS QR v10.10.7

## Universal EdgeTX package

This release contains both native EdgeTX entry points:

- Color widget: `/WIDGETS/GPSQR/main.lua`.
- Black-and-white/grayscale telemetry script: `/SCRIPTS/TELEMETRY/GPSQR.lua`.

Both produce the same `geo:<latitude>,<longitude>` payload and use unit-based GPS sensor discovery.

## Highlights

- Self-contained color widget using EdgeTX's native LVGL QR renderer.
- Responsive landscape, portrait, full-screen, and compact-zone layouts.
- Touch or physical-key manual refresh.
- Fixed Version 2-L encoder for radios without native color QR support.
- Cooperative Reed-Solomon and mask generation.
- Packed 32-bit QR matrix rows and cached rectangle runs.
- Renamed and multiple GPS sensor support.
- Last-known-position retention with fix-age status.
- Jumper T14 automatic inverted-polarity profile.
- Independent host QR decode verification.

## Validation

- Physically validated on RadioMaster TX16S.
- Host validated at 128×64 one-bit and 212×64 grayscale.
- Host validated at 480×272, 320×480, and 800×480 color layouts.

## Installation

Download `GPSQR-v10.10.7-SD-minified.zip`, extract it to the SD-card root, delete stale `main.luac` or `GPSQR.luac` files, and restart the radio.

See [`docs/INSTALLATION.md`](../INSTALLATION.md) for full instructions and [`docs/COMPATIBILITY.md`](../COMPATIBILITY.md) for validation scope.
